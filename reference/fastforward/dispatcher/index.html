    fastforward.dispatcher.register(op_name, predicate, kernel=None, priority=DispatcherPriority.DEFAULT) #

    register(op_name: str, predicate: Optional[Predicate[P]], kernel: KernelType[P], priority: DispatcherPriority = DispatcherPriority.DEFAULT) -> DispatcherRegistrationHook

    register(op_name: str, predicate: Optional[Predicate[P]], kernel: None = None, priority: DispatcherPriority = DispatcherPriority.DEFAULT) -> Callable[[KernelType[P]], KernelType[P]]

    Register a new implementation for an operator.

    This which will be used when the predicate evaluates to True. Might raise an exception if there is already a kernel with the same priority.

    register can be used standalone:

    register(op_name, predicate, kernel)

    as a standalone context manager:

    @register(op_name, predicate) def my_kernel(...): ...

    or using a with statement:

    > with register(op_name, predicate, kernel):
>     ...

    In the latter case, the registered implementation is removed at the end of the with block.
